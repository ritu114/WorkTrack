# WorkTrack : Outcome-Based Productivity Management System

## Overview
This project is a web-based productivity and performance management system designed to track work based on outcomes rather than attendance or activity. It supports role-based workflows for administrators, managers, and employees.

## Key Features
- Role-based authentication (Admin, Manager, Employee)
- Goal and task-based workflow
- Outcome-focused productivity measurement
- Team and individual performance dashboards
- Ethical and privacy-aware design

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MySQL / PostgreSQL
- Authentication: JWT
